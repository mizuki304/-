# 旅行サイト

ヨーロッパ旅行計画用のウェブアプリケーション

## 概要

友人グループのヨーロッパ旅行に必要な情報を一箇所に集約するためのサイトです。

## 機能

- 旅行日程の管理
- 予約情報の共有
- 予算管理
- 地図表示
- チェックリスト

## 開発メモ

- `.env` に `SITE_PASSWORD` を設定すると、全ページが共通パスワードで保護されます（`middleware.ts` で判定）。
- ローカル確認は `npm run dev` で http://localhost:3000 にアクセスし、`/auth` でパスワードを入力してください。

## 技術スタック

- Next.js
- React
- Vercel (デプロイ)
